# LifeDiary Analytics

LifeDiary Analytics - это ETL-пайплайн для автоматизированного сбора, структурирования и анализа записей из личных Telegram-дневников. Проект использует Telethon для безопасного доступа к данным и методы NLP для извлечения сущностей и метрик. Результаты сохраняются в реляционную базу данных (PostgreSQL), что позволяет проводить глубокий анализ с помощью SQL и визуализировать данные в Jupyter Notebook.

# Проблема и решение
Telegram часто используется как неформальная платформа для ведения личных заметок, но существующие инструменты (Obsidian, Notion) не всегда доступны. В результате ценные данные о привычках, эмоциях, сне и целях остаются несистематизированными и недоступными для анализа.

Проект парсит структурированные записи (сны, эмоции, цели, привычки), извлекает данные и сохраняет в БД. Аналитики выполняют SQL-запросы и строят визуализации в Jupyter для поиска паттернов вроде "не выспался, потому что мой эмоциональный фон был нарушен".
