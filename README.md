# LifeDiary Analytics

LifeDiary Analytics - это ETL-пайплайн для автоматизированного сбора, структурирования и анализа записей из личных Telegram-дневников. Проект использует Telethon для безопасного доступа к данным и методы NLP для извлечения сущностей и метрик. Результаты сохраняются в реляционную базу данных (PostgreSQL), что позволяет проводить глубокий анализ с помощью SQL и визуализировать данные в Jupyter Notebook.

# Проблема и решение
Пользователи активно используют Telegram для ведения личных записей, но эти данные остаются неструктурированными и недоступными для системного анализа. Существующие решения для ведения дневников не предоставляют инструментов для автоматизированной аналитики на основе естественного языка

# Решение
Система автоматически распознает и категоризирует типы записей: сны, эмоции, цели, привычки, активность. NLP-модуль извлекает количественные и качественные метрики из текстовых записей. Данные сохраняются в реляционную базу данных с четкой схемой для выполнения аналитических запросов. Аналитики могут строить визуализации и выявлять паттерны с использованием pandas, Seaborn и других инструментов анализа данных

# Пример применения
Пользователь ведет дневники в Telegram(telegram-каналы), отмечая ежедневные активность, сон и эмоциональное состояние. Пайплайн автоматически обрабатывает записи и загружает их в базу данных. Аналитик выполняет SQL-запросы для выявления корреляций между различными аспектами жизни. На основе полученных данных строятся дашборды и формируются insights для принятия решений
