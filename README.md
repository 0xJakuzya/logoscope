# LifeDiary Analytics

LifeDiary Analytics - end-to-end пайплайн для извлечения метрик жизни из приватных Telegram-дневников с помощью Telethon и NLP. Данные сохраняются в PostgreSQL для SQL-запросов и анализа в Jupyter Notebook.

# Проблема и решение
Большинство людей активно используют Telegram для разных целей, но не всегда есть доступ к Obsidian, Notion или блокнотам. LifeDiary позволяет вести дневники прямо в вкладке "Избранное" - просто и удобно.​

Проект парсит структурированные записи (сны, эмоции, цели, привычки), извлекает метрики и сохраняет в БД. Аналитики выполняют SQL-запросы и строят визуализации в Jupyter для поиска паттернов вроде "не выспался, потому что мой эмоциональный фон был нарушен".

# Для кого
- Начинающие/опытные data analysts: практика SQL, ETL, pandas/Seaborn, ML 
- Помешанные на self-tracking: самодисциплина в заполнении = богатые датасеты для анализа.

Убиваем 2-ух зайцев: удобный дневник + симулятор аналитика на личных данных. Локально, без облаков.
